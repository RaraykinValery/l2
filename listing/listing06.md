Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3, 2, 3]

append в функции modifySlice создаёт новый массив, так как cap массива i = 3
и все эти места уже заняты. Таким образом после первого append в функции операция
замены значения по индексу будет происходить совсем с другим массивом.
```
